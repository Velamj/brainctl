# Code of Conduct

## Our Standard

Be respectful. Be constructive. Ship good code.

We welcome contributions from everyone regardless of experience level, background, or identity.

## Expected Behavior

- Be direct and constructive in code reviews
- Focus on the work, not the person
- Help newcomers get oriented
- Credit others' contributions

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Trolling or deliberately inflammatory comments
- Publishing others' private information

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code. Repeated violations result in a ban.

Report issues to the maintainers via GitHub issues or email.

## Scope

This applies to all project spaces — issues, PRs, discussions, and any public representation of the project.
